Raspberry PiにDockerをインストールする

Posted on September 24, 2021 by nobiruwa

切っ掛け

mozilla/TTSがpip install ttsだけではインストールできなかったため、有志が公開しているdockerイメージを使ってみたいと思いました。

手順はHow to Install Docker on Raspberry Piを参考にしました。

手順

get-docker.shをダウンロードします。

$ curl -fsSL https://get.docker.com -o get-docker.sh

root権限でget-docker.shを実行します。リポジトリの登録と、パッケージ(docker-ce, docker-ce-cli, docker-ce-rootless-extras)のインストールが行われます。

一般ユーザー権限でdockerd-rootless.shを実行します。

$ sudo apt install uidmap
$ dockerd-rootless.sh install

インストール後

試しにsynesthesiam/docker-mozillattsを動作させます。

$ docker run -it -p 5002:5002 synesthesiam/mozillatts:en
